  2. Start the angular-cli live dev server (npm start).
  3. Load the page.
  4. Stop the vortexpy server.
  5. Reload the page (the page will load but the vortex will be offline.
  6. CHECK - You will see the olddata load on the page
  7. Start the vortexpy server
  8. The vortex will reconnect automatically (DO NOT RELOAD)
  9. CHECK - The data should update with the latest data from the server.
